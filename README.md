The main aim of the project is to build a collaborative web platform which allows programming language students to keep an online journal. Our project is all about to help the people who are unable to crack their jobs in programming area. Our blog is a journal that is frequently updated and intended for general public consumption. Any blogs are popular just because it allows people to interact with each other. The ability of readers to leave their comments in an interactive format is an important part of our blogs. We have developed this several blogs for ensuring effective policing by providing statistics of the members. The MySQL database is used as a platform along with WAMP Server support. Application are developed in HTML5, CSS3, JavaScript using FLASK as an integration system. Overall this Classic Blog is all about to manage the programming related activities.
